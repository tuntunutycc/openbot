# Roadmap

This roadmap is synchronized to feature folders under `specs/`.

## Feature Milestones

### openclaw_init
- Status: Done
- Scope: OpenClaw runtime bootstrap, agent wiring, and `/agent` routing.

### photoroom_api
- Status: Done
- Scope: Background removal pipeline and Telegram image/document handling.

### ai_pipeline
- Status: Done
- Scope: Anthropic parsing + Photoroom edit orchestration and fallback behavior.

### catalog_generation
- Status: Done
- Scope: Catalog composition pipeline and `/catalog` caption-triggered routing.

### batch_variations
- Status: Done
- Scope: Multi-variation catalog generation and album delivery flow.

### custom_prompts
- Status: Done
- Scope: Burmese/English prompt enhancement and distinct catalog variation prompting.

### dynamic_image_editing
- Status: Done
- Scope: `/edit` dynamic image editing flow with mapping + validation safeguards.

## Notes
- Feature names are intentionally identical to `specs/<feature_name>/` directory names.
- Legacy phase labels remain in `changelog.md` history entries for traceability.
